## Image Grid Layout

A responsive image grid layout built using <b>HTML</b> and <b>CSS Grid.</b>
This project demonstrates how to create a clean, centered, flexible grid of images arranged in a 3x2 layout, with each image displayed inside a card-style container.

## Github Repository
https://github.com/anerimoradia/image-grid.git

## Project Objective
<b> CSS Grid Structure </b>
<ul>
    <li>Create a grid container using <blockquote>display:grid</blockquote></li>
    <li> Define columns and gaps </li>
    <li> Place six images inside grid items </li>
</ul>
